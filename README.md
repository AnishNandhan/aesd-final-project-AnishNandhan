# Overview

A Yocto Project layer that builds and packages a minimal Type-2 hypervisor
for the Raspberry Pi 4 Model B. The layer provides a KVM-enabled kernel
configuration, a userspace VMM, a bare-metal ARM64 guest binary, and a small
virtio-blk disk image, all assembled into a bootable image that runs a
hardware-isolated virtual machine from a single command.
